== Description ==

This latest referrers plugin allows you to display the recent websites your visitors have found you from - Through a widget on your blog!

Simply select the title of the widget and the number of referrals you wish to show, enable the widget and then everything will be active!

== Installation ==

1. Upload `wp referrer.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Edit the Referrer WP Settings in your Administration Panel
4. Enable the Widget through the Appearance --> Widgets tab.
